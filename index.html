About us

Llamita vet is a mobile application where the main function is to have control of your pet. This is achieved with the application as it allows you to save all your pet's information and update it.

With the intention of giving each pet a better life, with better care, better professionals where to be attended. Llamita vet provides you with tools that will make you feel that your pet matters.
